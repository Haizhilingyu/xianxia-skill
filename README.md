# 仙侠技能 (Xianxia Skill)

> 修仙知识管理系统 - OpenClaw 独立技能模块

## 简介

这是一个为 OpenClaw 设计的仙侠主题个人知识管理技能，融合 Zettelkasten 卡片笔记法与修仙体系。

**核心特性**：
- 🏠 **独立自包含**：不依赖外部系统，可独立运行
- 🚀 **一键初始化**：可自动创建完整的仙侠系统
- 📦 **完整模板库**：9个自包含模板（5笔记+4日志）
- 🔗 **智能链接**：自动检测主题关联，建立双向链接
- 🏆 **修炼体系**：境界提升、成就系统、修炼值奖励

## 核心特性

### 🔄 主题关联检测

炼化灵石时自动检测主题关联，建立双向链接：

- **自动检测**：分析灵石关键词，匹配已有灵石
- **关联强度**：
  - 强关联（≥3关键词）→ 自动建立双向链接
  - 中等关联（1-2关键词）→ 评估后建立链接
  - 弱关联（主题相关）→ 记录待观察
- **阵法触发**：3+灵石同一主题 → 自动创建阵法核心

### 🔗 双向链接

建立知识网络的关键机制：

```markdown
<!-- 在灵石A中 -->
#### 连接的灵石
- [[灵石B]]

<!-- 在灵石B中 -->
#### 连接的灵石
- [[灵石A]]
```

**链接奖励**：
- 创建链接：+2 修炼值
- 被链接：+5 修炼值
- 阵法核心：+50 修炼值

## 独立性说明

**仙侠技能是完全独立的**：
- ✅ 不依赖任何外部仙侠系统
- ✅ 包含所有必要的模板和参考资料
- ✅ 可在空目录中初始化完整系统
- ✅ 可与现有仙侠系统无缝集成

**初始化方式**：
```bash
# 方式1：自动检测
首次使用时，技能会自动检测 ~/xianxia/ 是否存在
如果不存在，会提示是否初始化

# 方式2：手动触发
对驻守说："初始化仙侠系统"或"创建仙侠系统"
```

**初始化后会创建**：
```
~/xianxia/
├── 000混沌之初/
├── 100秩序始端/{核心概念,实践应用,个人思考,疑难解答}
├── 300修身世界/
├── 400术法专攻/
└── 999系统外挂/{模板,修炼系统,使用指南,日记}
    ├── README.md
    └── 修炼系统/我的修炼状态.md
```

## 使用方式

将此技能放置在 `~/.openclaw/skills/xianxia/` 目录下，OpenClaw 会自动加载。

### 触发词

- `"混沌之气"` / `"记一下"` / `"写个笔记"` → 记录混沌之气
- `"炼化"` / `"炼化成灵石"` → 炼化混沌之气
- `"查询灵石"` / `"查知识库"` → 搜索知识库
- `"创建阵法"` → 创建阵法核心
- `"传世之作"` → 创建传世之作
- `"修炼状态"` / `"我的境界"` → 查看修炼状态
