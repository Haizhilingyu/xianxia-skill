# 仙侠技能 (Xianxia Skill)

> 修仙知识管理系统 - OpenClaw 技能模块

## 简介

这是一个为 OpenClaw 设计的仙侠主题个人知识管理技能，融合 Zettelkasten 卡片笔记法与修仙体系。

## 核心概念

| 术语 | 概念 | 说明 |
|------|------|------|
| **混沌之气** | Fleeting Notes | 临时笔记，每天炼化清空 |
| **灵石** | Permanent Notes | 永久知识，原子化 |
| **秘籍残页** | Literature Notes | 学习笔记，阅读摘录 |
| **阵法核心** | Theme Notes | 主题知识网络 |
| **传世之作** | Articles | 输出文章 |
| **炼化日志** | Daily Log | 每日炼化统计 |

## 每日修炼流程

```
白天捕捉 → 晨间炼化 → 晚间链接 → 每周阵法 → 每月产出
   ↓           ↓          ↓          ↓          ↓
混沌之气    灵石笔记    建立连接   阵法核心   传世之作
```

| 筑基期 | 1000-5000 | 创建100个灵石笔记 |

| 大乘期 | 500000-1000000 | 输出影响他人 |
| 渡劫期 | 1000000+ | 创造传世经典 |

## 使用方式

将此技能放置在 `~/.openclaw/skills/xianxia/` 目录下，OpenClaw 会自动加载。

### 触发词

- `"混沌之气"` / `"记一下"` / `"写个笔记"` → 记录混沌之气
- `"炼化"` / `"炼化成灵石"` → 炼化混沌之气
- `"查询灵石"` / `"查知识库"` → 搜索知识库
- `"创建阵法"` → 创建阵法核心
- `"传世之作"` → 创建传世之作
- `"修炼状态"` / `"我的境界"` → 查看修炼状态

## 依赖

- OpenClaw 运行环境
- 目标知识库目录：`~/xianxia/`
