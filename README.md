# 仙侠技能 (Xianxia Skill)

> 修仙知识管理系统 - OpenClaw 技能模块

## 简介

这是一个为 OpenClaw 设计的仙侠主题个人知识管理技能，融合 Zettelkasten 卡片笔记法与修仙体系。

## 核心概念

| 术语 | 概念 | 说明 |
|------|------|------|
| **混沌之气** | Fleeting Notes | 临时笔记，每天炼化清空 |
| **灵石** | Permanent Notes | 永久知识，原子化 |
| **秘籍残页** | Literature Notes | 学习笔记，阅读摘录 |
| **阵法核心** | Theme Notes | 主题知识网络 |
| **传世之作** | Articles | 输出文章 |
| **修炼日记** | Daily Log | 每日修炼记录（999系统外挂/日记/） |

## 核心特性

### 🔄 主题关联检测

炼化灵石时自动检测主题关联，建立双向链接：

- **自动检测**：分析灵石关键词，匹配已有灵石
- **关联强度**：
  - 强关联（≥3关键词）→ 自动建立双向链接
  - 中等关联（1-2关键词）→ 评估后建立链接
  - 弱关联（主题相关）→ 记录待观察
- **阵法触发**：3+灵石同一主题 → 自动创建阵法核心

### 🔗 双向链接

建立知识网络的关键机制：

```markdown
<!-- 在灵石A中 -->
#### 连接的灵石
- [[灵石B]]

<!-- 在灵石B中 -->
#### 连接的灵石
- [[灵石A]]
```

**链接奖励**：
- 创建链接：+2 修炼值
- 被链接：+5 修炼值
- 阵法核心：+50 修炼值

## 每日修炼流程

```
白天捕捉 → 晨间炼化 → 晚间链接 → 每周阵法 → 每月产出
   ↓           ↓          ↓          ↓          ↓
混沌之气    灵石笔记    建立连接   阵法核心   传世之作
```

## 使用方式

将此技能放置在 `~/.openclaw/skills/xianxia/` 目录下，OpenClaw 会自动加载。

### 触发词

- `"混沌之气"` / `"记一下"` / `"写个笔记"` → 记录混沌之气
- `"炼化"` / `"炼化成灵石"` → 炼化混沌之气
- `"查询灵石"` / `"查知识库"` → 搜索知识库
- `"创建阵法"` → 创建阵法核心
- `"传世之作"` → 创建传世之作
- `"修炼状态"` / `"我的境界"` → 查看修炼状态
