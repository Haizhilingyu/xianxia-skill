# 修仙知识管理系统

建平儿的仙侠主题个人知识管理系统，融合 Zettelkasten 卡片笔记法。

## 系统核心概念

| 术语 | 概念 | 位置 | 说明 |
|------|------|------|------|
| **混沌之气** | Fleeting Notes | `000混沌之初/` | 临时笔记，**每天炼化清空** |
| **灵石** | Permanent Notes | `100秩序始端/` | 永久知识，原子化 |
| **秘籍残页** | Literature Notes | `100秩序始端/` | 学习笔记，阅读摘录 |
| **阵法核心** | Theme Notes | `100秩序始端/` | 主题知识网络 |
| **传世之作** | Articles | 各领域 | 输出文章 |
| **炼化日志** | Daily Log | `memory/` | 每日炼化统计 |

```bash

read ~/.openclaw/skills/xianxia/templates/article.md    # 创建传世之作
read ~/.openclaw/skills/xianxia/templates/refining-log.md # 炼化日志
```

## 1. 记录混沌之气

**目标**：`~/xianxia/000混沌之初/`

**流程**：
1. 提取标题，生成文件名：`标题-YYYYMMDDHHmmss.md`
2. **读取模板**：`read templates/chaos.md`
3. 填充内容，使用 write 工具保存
4. 返回确认信息

**文件名规则**：移除特殊字符 `\/:*?"<>|`，空格替换为连字符

---

## 2. 炼化混沌之气（核心流程）

**目标**：根据内容类型选择目录

| 内容类型 | 目标目录 |
|---------|---------|
| 技术相关 | `400术法专攻/` |
| 生活相关 | `300修身世界/` |
| 通用知识 | `100秩序始端/核心概念/` |
| 实践经验 | `100秩序始端/实践应用/` |
| 个人见解 | `100秩序始端/个人思考/` |
| 问题解决 | `100秩序始端/疑难解答/` |

**流程**：
1. 读取混沌之气内容
2. 确定目标目录
3. **提炼核心观点**：一个灵石 = 一个观点
4. 使用 **灵石模板**（见 `templates/spirit.md`）
5. **处理原混沌之气**：
   - 方式A：在 frontmatter 中添加 `status: 已炼化`
   - 方式B：移动到 `000混沌之初/已炼化/`
6. **记录炼化日志**（见 `templates/refining-log.md`）

## 4. 查询功能

### 4.1 查询当日混沌之气
**触发词**：`"今天记录了什么"` / `"混沌之气有哪些"`

**执行命令**：
```bash
ls -lt ~/xianxia/000混沌之初/*.md 2>/dev/null | grep -v "已炼化" | head -20
```

### 4.2 查询灵石（知识库搜索）
**触发词**：`"查询灵石"` / `"查知识库"` / `"搜索知识"`

**执行命令**：
```bash
# 按关键词搜索（使用 exec 工具）
grep -ril "关键词" ~/xianxia/100秩序始端/ ~/xianxia/300修身世界/ ~/xianxia/400术法专攻/

# 按标签搜索
grep -ril "#标签" ~/xianxia/

# 列出某目录所有文件
ls ~/xianxia/100秩序始端/核心概念/
```

### 4.3 查询炼化日志
**触发词**：`"查看炼化日志"`

**执行命令**：
```bash
ls -lt ~/xianxia/memory/*.md | head -10
```

## 7. 修炼状态追踪

**触发词**：`"修炼状态"` / `"我的境界"`

**读取文件**：`~/xianxia/999系统外挂/修炼系统/我的修炼状态.md`

**修炼值计算**：
```
总修炼值 = Σ(笔记 cultivation_value) + 链接奖励
链接奖励 = (出链接数 × 2) + (入链接数 × 5)
```

**自动更新时机**：
| 操作 | 更新内容 |
|------|---------|
| 炼化混沌之气后 | 总修炼值 +10 |
| 创建阵法核心后 | 总修炼值 +50，阵法数量 +1 |
| 发布传世之作后 | 总修炼值 +100，文章数量 +1 |
| 建立链接后 | 链接奖励 +2/个 |
| 每日结束时 | 计算当日总修炼值，更新境界进度 |

# 边缘情况处理

| 情况 | 处理方式 |
|------|---------|
| 标题为空 | 使用 `混沌之气-时间戳` |
| 文件已存在 | 追加内容（添加分隔线） |
| 混沌之气为空 | 提示"没有待炼化的混沌之气" |
| 查询无结果 | 提示"未找到，试试其他关键词" |

# 错误处理

## 文件操作错误

| 错误情况 | 处理方式 |
|---------|---------|
| 目录不存在 | `mkdir -p ~/xianxia/目录名` 创建目录 |
| 文件写入失败 | 检查权限，提示用户 |
| 文件读取失败 | 提示"文件不存在或无法读取" |
| 磁盘空间不足 | 提示用户清理空间 |

## 搜索错误

| 错误情况 | 处理方式 |
|---------|---------|
| grep 无匹配 | 返回"未找到相关内容"而非报错 |
| 目录为空 | 提示"该目录下没有文件" |
| 权限不足 | 提示"无法访问该目录" |

## 炼化错误

| 错误情况 | 处理方式 |
|---------|---------|
| 混沌之气文件损坏 | 尝试恢复或提示用户手动检查 |
| 目标目录无法确定 | 询问用户选择目录 |
| 内容无法提炼 | 提示"内容过于模糊，请补充详情" |

## 模板错误

| 错误情况 | 处理方式 |
|---------|---------|
| 模板文件丢失 | 使用内置简化模板 |
| 模板格式错误 | 跳过模板，直接创建文件 |
