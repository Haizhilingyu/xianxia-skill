## 2. 炼化混沌之气（核心流程）

**目标**：根据内容类型选择目录

| 内容类型 | 目标目录 |
|---------|---------|
| 技术相关 | `400术法专攻/` |
| 生活相关 | `300修身世界/` |
| 通用知识 | `100秩序始端/核心概念/` |
| 实践经验 | `100秩序始端/实践应用/` |
| 个人见解 | `100秩序始端/个人思考/` |
| 问题解决 | `100秩序始端/疑难解答/` |

**流程**：
1. 读取混沌之气内容
2. 确定目标目录
3. **提炼核心观点**：一个灵石 = 一个观点
4. 使用 **灵石模板**（见 `templates/spirit.md`）
5. **检测主题关联**：
   - 分析新灵石与已有灵石的主题相关性
   - 如果发现关联，在两个灵石中互相添加 `[[灵石名称]]` 链接
   - 如果多个灵石属于同一主题，创建或更新阵法核心
6. **处理原混沌之气**：
   - 方式A：在 frontmatter 中添加 `status: 已炼化`
   - 方式B：移动到 `000混沌之初/已炼化/`
7. **记录修炼日记**

### 主题关联检测规则

**检测时机**：每炼化一个灵石后

**关键词提取方法**：
1. 从标题中提取名词和动词（排除停用词：的、了、在、是等）
2. 从核心观点中提取重复出现的词汇（≥2次）
3. 从标签中提取主题词
4. 合并去重，得到关键词列表

**关键词提取示例**：
```markdown
标题："OpenClaw配置修改注意事项"
核心观点："修改OpenClaw配置时需要注意权限问题"
标签：#技术/OpenClaw/配置

提取关键词：OpenClaw、配置、修改、权限、注意事项
```

**检测方法**：
1. 提取新灵石的关键词和主题
2. 在知识库中搜索相关灵石（使用 grep 搜索关键词）
3. 计算关键词匹配度

**关联强度判断**：
- **强关联**：关键词匹配 ≥ 3 个 → **自动建立双向链接**
- **中等关联**：关键词匹配 1-2 个 → 评估主题相关性后决定
- **弱关联**：主题相关但关键词不匹配 → 记录待观察，不强制建立链接

**中等关联评估示例**：
```markdown
❌ 不建立链接的情况：
灵石A："OpenClaw配置修改"（关键词：OpenClaw、配置）
灵石B："Git提交规范"（关键词：Git、提交）
判断：虽然都涉及"修改/提交"动作，但主题不相关 → 不建立链接

✅ 建立链接的情况：
灵石A："OpenClaw配置修改"（关键词：OpenClaw、配置）
灵石B："OpenClaw安装指南"（关键词：OpenClaw、安装）
判断：都有"OpenClaw"关键词，主题相关 → 建立链接
```

**双向链接格式**：
```markdown
<!-- 在灵石A中 -->
#### 连接的灵石
- [[灵石B]]

<!-- 在灵石B中 -->
#### 连接的灵石
- [[灵石A]]
```

**阵法核心触发**：
- 当发现 3+ 个灵石属于同一主题时
- 创建或更新阵法核心，将这些灵石链接起来
- 在每个灵石中添加 `[[阵法核心]]` 链接

### 链接维护机制

**灵石重命名时**：
1. 检测是否有其他灵石引用该灵石
2. 如果有，更新所有 `[[旧名称]]` 为 `[[新名称]]`
3. 更新修炼日记中的引用

**灵石删除时**：
1. 检查是否有其他灵石引用该灵石（grep 搜索 `[[灵石名称]]`）
2. 如果有引用：
   - 提示用户："有 X 个灵石引用此灵石，是否继续删除？"
   - 如果用户确认，删除所有指向该灵石的链接
3. 更新修炼日记

**链接失效检测**：
- 定期（每周）检查所有双向链接的有效性
- 发现失效链接时，记录到"待修复链接"列表
- 提示用户处理

**链接更新命令**：
```bash
# 搜索引用某灵石的所有文件
grep -r "\[\[灵石名称\]\]" ~/xianxia/

# 批量替换链接名称
sed -i 's/\[\[旧名称\]\]/[[新名称]]/g' ~/xianxia/**/*.md
```

# 错误处理

## 文件操作错误

| 错误情况 | 处理方式 |
|---------|---------|
| 目录不存在 | `mkdir -p ~/xianxia/目录名` 创建目录 |
| 文件写入失败 | 检查权限，提示用户 |
| 文件读取失败 | 提示"文件不存在或无法读取" |
| 磁盘空间不足 | 提示用户清理空间 |

## 搜索错误

| 错误情况 | 处理方式 |
|---------|---------|
| grep 无匹配 | 返回"未找到相关内容"而非报错 |
| 目录为空 | 提示"该目录下没有文件" |
| 权限不足 | 提示"无法访问该目录" |

## 炼化错误

| 错误情况 | 处理方式 |
|---------|---------|
| 混沌之气文件损坏 | 尝试恢复或提示用户手动检查 |
| 目标目录无法确定 | 询问用户选择目录 |
| 内容无法提炼 | 提示"内容过于模糊，请补充详情" |

## 模板错误

| 错误情况 | 处理方式 |
|---------|---------|
| 模板文件丢失 | 使用内置简化模板 |
| 模板格式错误 | 跳过模板，直接创建文件 |
