# 修炼境界体系

## 境界划分

| 境界 | 修炼值要求 | 突破条件 | 特权 |
|------|-----------|---------|------|
| **炼气期** | 0-1000 | 每100值升一层（9层） | 基础笔记功能 |
| **筑基期** | 1000-5000 | 创建100个灵石笔记 | 可创建阵法核心 |
| **金丹期** | 5000-20000 | 创建10个主题阵法 | 可创建传世之作 |
| **元婴期** | 20000-50000 | 发布5篇传世之作 | 阵法自动关联 |
| **化神期** | 50000-100000 | 形成知识体系闭环 | 跨域知识融合 |
| **炼虚期** | 100000-200000 | 跨领域融合创新 | 自动提炼主题 |
| **合体期** | 200000-500000 | 知识复用率>50% | 智能推荐链接 |
| **大乘期** | 500000-1000000 | 输出影响他人 | 自动生成传世之作 |
| **渡劫期** | 1000000+ | 创造传世经典 | 飞升成仙 |

## 境界突破天劫

### 炼气期 → 筑基期
**修炼值要求**：1000
**突破条件**：
- 创建100个灵石笔记
- 形成至少5个稳定的主题
- 建立至少20个双向链接

**天劫任务**：
- 选择最熟悉的领域
- 创建阵法核心
- 连接至少30个灵石
- 形成完整的知识脉络

### 筑基期 → 金丹期
**修炼值要求**：5000
**突破条件**：
- 创建10个阵法核心
- 每个阵法连接≥20个灵石
- 在一个领域形成系统性认知

**天劫任务**：
- 深度研究一个主题
- 阅读相关书籍/资料10+
- 创建100+灵石笔记
- 形成独特见解

### 金丹期 → 元婴期
**修炼值要求**：20000
**突破条件**：
- 发布5篇传世之作
- 每篇≥2000字
- 获得外部反馈

**天劫任务**：
- 选择最有价值的知识
- 系统整理成课程/文章
- 发布并获得反馈
- 帮助至少10人学习

### 元婴期 → 化神期
**修炼值要求**：50000
**突破条件**：
- 形成知识体系闭环
- 跨领域建立关联
- 创造新的知识组合

**天劫任务**：
- 识别两个不同领域的共同点
- 创造新的连接或方法
- 验证新方法的有效性
- 整理成可传播的知识

## 修炼值获取规则

### 基础修炼值

| 操作 | 修炼值 | 说明 |
|------|--------|------|
| 创建混沌之气 | +1 | 快速捕捉灵感 |
| 炼化成灵石 | +10 | 提炼核心观点 |
| 创建秘籍残页 | +5 | 学习笔记 |
| 创建阵法核心 | +50 | 主题知识网络 |
| 创建传世之作 | +100 | 输出文章 |

### 链接奖励

| 操作 | 修炼值 | 说明 |
|------|--------|------|
| 创建出链接 | +2 | 每个链接 |
| 被链接 | +5 | 每个入链接 |
| 阵法连接灵石 | +2 | 每个灵石 |

### 额外奖励

| 操作 | 修炼值 | 说明 |
|------|--------|------|
| 每日炼化≥10混沌之气 | +20 | 勤奋修炼 |
| 连续7天每日创建≥3灵石 | +50 | 持之以恒 |
| 连续30天每日修炼 | +200 | 坚持不懈 |
| 成就解锁 | +10~1000 | 视成就等级 |

## 境界进度计算

```python
def get_realm_progress(cultivation_value):
    if value < 1000:
        realm = "炼气期"
        layer = value // 100 + 1
        progress = (value % 100) / 100 * 100
        return f"{realm}{layer}层 ({value}/1000) - 进度 {progress}%"
    elif value < 5000:
        progress = (value - 1000) / 4000 * 100
        return f"筑基期 ({value}/5000) - 进度 {progress}%"
    elif value < 20000:
        progress = (value - 5000) / 15000 * 100
        return f"金丹期 ({value}/20000) - 进度 {progress}%"
    # ... 以此类推
```

## 修炼值计算公式

```
总修炼值 = Σ(笔记 cultivation_value) + 链接奖励 + 成就奖励 + 活动奖励

链接奖励 = (出链接数 × 2) + (入链接数 × 5)
活动奖励 = 每日炼化奖励 + 连续修炼奖励
```
