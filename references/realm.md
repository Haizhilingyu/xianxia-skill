# 修炼境界体系

## 境界划分

| 境界 | 修炼值要求 | 突破条件 |
|------|-----------|---------|
| 炼气期 | 0-1000 | 每100值升一层（9层） |
| 筑基期 | 1000-5000 | 创建100个灵石笔记 |
| 金丹期 | 5000-20000 | 创建10个主题阵法 |
| 元婴期 | 20000-50000 | 发布5篇传世之作 |
| 化神期 | 50000-100000 | 形成知识体系闭环 |
| 炼虚期 | 100000-200000 | 跨领域融合创新 |
| 合体期 | 200000-500000 | 知识复用率>50% |
| 大乘期 | 500000-1000000 | 输出影响他人 |
| 渡劫期 | 1000000+ | 创造传世经典 |

## 修炼值获取规则

### 基础修炼值
| 操作 | 修炼值 |
|------|--------|
| 混沌之气 | +1 |
| 灵石笔记 | +10 |
| 秘籍残页 | +5 |
| 阵法核心 | +50 |
| 传世之作 | +100 |

### 额外奖励
| 操作 | 修炼值 |
|------|--------|
| 创建链接 | +2/个 |
| 被链接 | +5/个 |
| 完善已有笔记 | +3 |
| 每日炼化≥10混沌之气 | +20 |
| 连续7天每日创建≥3灵石 | +50 |

## 境界突破天劫

### 筑基天劫
- 创建100个灵石笔记
- 形成10个核心概念的知识树
- 每棵知识树连接≥5个灵石

### 金丹天劫
- 创建10个阵法核心
- 每个阵法连接≥20个灵石
- 在一个领域形成系统性认知

### 元婴天劫
- 发布5篇传世之作
- 每篇≥2000字
- 获得外部反馈

## 修炼值计算

```
总修炼值 = Σ(笔记 cultivation_value) + 链接奖励

链接奖励 = (出链接数 × 2) + (入链接数 × 5)
```
